# ansible_roles
Simple Ansible Roles

### Commands

- Initialize a role
```
ansible-galaxy init role1
```

### role1 tree structure
```
role1/
├── defaults
│   └── main.yml
├── files
│   └── testfile
├── handlers
│   └── main.yml
├── meta
│   └── main.yml
├── README.md
├── tasks
│   └── main.yml
├── templates
├── tests
│   ├── inventory
│   └── test.yml
└── vars
    └── main.yml
```
